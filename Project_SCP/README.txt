Vous trouverez Trois dossiers dans ce reperatoire :
----------------------------------------------------

	* Livrables :
		Contenant le rapport.pdf et La map de notre jeu ainsi que les images en meilleur qualité pour l'UML , diagramme de séquence et d'état.

	* POO_Project_G.7_eclipse :
		Ceci une version du projet sous eclipse, il suffit de l'installer sur celui-ci, et d'ouvrir le 
	    package game puis le fichier SCP_CB et lancer le jeu .

	* POO_Project_G.7_Linux :	
		Il s'agit ici d'une version sous linux avec un Makefile, ainsi pour compiler les fichiers vous devrez  
	    lancer la commande : make all , ensuite pour lancer le jeu il suffit de lancer la commande : make start , finalement
	    si vous souhaitez supprimer les fichier .classe, lancez la commande : make clean.
	  

	        Ps : les fichiers de tests ne sont pas inclus à la compilation dans le makefile !
	 

Comment jouer ? 
----------------

		Une fois le jeu installé et lancé, vous êtes perdu dans une salle obscure. Vous entendez des bruits, des hurlements de partout mais 
	vous entendez surtout le bruit d'une alarme. Vous avez peur, votre instinct comprend que vous devez vous échapper d'ici le plus rapidement 
	possible en relevant quelques défis et sans mourir face aux différents obstacles. Vous pourrez utiliser la commande "help" dans le jeu pour
	connaître toutes les actions et commandes possibles. 

	Bonne chance aventurier.
